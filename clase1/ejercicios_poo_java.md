# Ruta de ejercicios POO en Java

## Nivel básico

### Ejercicio 1: Perfil Gamer

Enunciado:
En una plataforma de videojuegos, cada jugador tiene un perfil con la siguiente información:
- Nombre de usuario
- Juego favorito
- Nivel
- Cantidad de partidas ganadas

Instrucciones:
1. Ubica la carpeta del proyecto en VS Code.
2. Busca la carpeta src.
3. Haz click derecho sobre src y selecciona "New File".
4. Nombra el archivo Gamer.java. Nota: la extensión .java no es necesario escribirla Visual Studio la escribirá por ti.


Actividades:
- Crea la estructura básica de una clase.
- Define los atributos.

Sugerencias:
- ¿Qué tipo de dato usarías para texto?
- ¿Qué tipo de dato usarías para números?
- Usa modificador private.

## Nivel medio-alto

### Ejercicio 5: Post

Enunciado:
Una red social guarda publicaciones:
- Usuario
- Contenido
- Likes
- Comentarios

Instrucciones:
Crear archivo Post.java dentro de src.

Actividades:
- Crear la clase.
- Definir atributos privados.
- Crear constructor.
- Crear getters.

Reto:
- Crear un método para aumentar los likes.
- Intanscia la clase en el método main de la clase principal y aumenta 5 veces los likes
- Imprimir por consola la cantidad de likes

Sugerencias:
- ¿Por qué no usar atributos públicos?
- ¿Qué acción representa un método?
